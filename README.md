# Obstacle Dodge

---

This is my first step into the world of game development! this project has been created to start in the Game Dev Loop.

---

## About the Project

Obstacle Dodge is a simple yet functional arcade-style game set in the deep ocean. You take control of a brave hermit crab that must dodge obstacles to survive in the underwater world.

## Learning Objectives

* Understanding 3D coordinate systems and movement.
* Implementing collision logic and "Game Over" states.
* Creating custom shaders to achieve a unique visual identity.
* Configuring underwater atmosphere (Fog, Post-processing, and Particle Systems).

---

## Built With

| Category | Technology |
| :--- | :--- |
| **Engine** | Unity 2022.3+ (URP - Universal Render Pipeline) |
| **Language** | C# for movement logic and triggers |
| **Shaders** | Custom Shader Graph for the sand and water effects |
| **VFX** | Unity Particle System for "Marine Snow" suspension |
